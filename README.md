# Agro Forte

Jogo educativo sobre sustentabilidade no agronegócio.

---

## 🚀 Objetivo

O principal desafio do jogo é **equilibrar a produção agrícola e a preservação ambiental**. O jogador precisa gerenciar seus recursos financeiros enquanto garante que suas ações não destruam o ecossistema.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação da interface e dos elementos visuais.
* **CSS3:** Estilização, layout responsivo e animações das barras de status.
* **JavaScript (ES6):** Lógica do jogo, controle de pontuação e mecânicas de vitória/derrota.
